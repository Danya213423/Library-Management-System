#Library Management System (Console App)

Консольное приложение на **C#** для управления личной библиотекой книг. Проект демонстрирует навыки работы с объектно-ориентированным программированием, коллекциями и внешними библиотеками в экосистеме **.NET**

#Основные возможности:
*   Управление данными (CRUD): Добавление, просмотр детальной информации и удаление книг.
*   Умное удаление: Система распознает книги с одинаковыми названиями и предлагает выбрать конкретную по индексу.
*   Поиск: Быстрый поиск по названию книги (регистронезависимый).
*   Автономность:Данные автоматически сохраняются в JSON-файл и подгружаются при старте
*   (используется библиотека `Newtonsoft.Json`).
*   Валидация: Проверка на пустые поля при вводе и защита от ошибок (инпутов) при выборе пунктов меню.
  
#Стек технологий:
*   Язык: C# (.NET)
*   Библиотеки: Newtonsoft.Json
*   Инструменты: LINQ (фильтрация и поиск), File I/O (работа с файловой системой)

#Как запустить:
1. Клонируйте репозиторий.
2. Откройте файл решения в Visual Studio.
3. Соберите проект и запустите.
